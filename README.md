# Reverse Shell
Node.js reverse shell using socket.io & readcommand

## Requirements
7.0.0 <= Node.js <br>
0.3.0 <= readcommand <br>
2.0.1 <= socket.io <br>
2.0.1 <= socket.io-client <br>
0.0.1 <= fs <br>
0.0.0 <= http
## Install
```bash
$ git clone https://github.com/MaherBTA/reverse-shell
$ cd reverse-shell
$ npm install
```

## Run
```bash
$ node server.js
open browser http://localhost:80
```
